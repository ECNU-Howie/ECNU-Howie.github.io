所需工具

node.js + Git + Hexo + Github

一些命令：

命令手册：https://hexo.io/zh-cn/docs/commands.html#new

注：（这些命令最好在博客路径运行）

  • 初始化博客：hexo init 文件夹名称
  • 新建文章：hexo new 文章名称(或者)hexo n 文章名称
  • 清除缓存：hexo clean
  • 生成静态文件：hexo generate（或者）hexo g
  • 在线预览：hexo server(或者)hexo s
  • 一键部署：hexo deploy(或者)hexo d
